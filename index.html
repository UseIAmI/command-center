Setup Guide
Step 1: GAS → Extensions → Apps Script → Project Settings → Script Properties
  Add: GEMINI_API_KEY · APOLLO_API_KEY · APOLLO_SEQUENCE_ID · ADMIN_EMAIL · COMMAND_CENTER_PW · SUBSTACK_EMAIL · VAULT_FOLDER_ID

Step 2: Deploy GAS as Web App → Execute as Me → Anyone can access
  Copy the Web App URL and set it as GAS_ENDPOINT in this file

Step 3: Deploy this file to Netlify → Set custom domain: commandcenter.useiami.com
  Add CNAME DNS record: commandcenter → [your-netlify-subdomain].netlify.app

Step 4: Run "⚙ Setup 8AM Auto-Trigger" from the GAS menu in your Google Sheet
